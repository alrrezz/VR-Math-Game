# Math Swords VR
**Slash Your Way Through Math – Dual Sword VR Arcade!**

![Math Swords VR](https://via.placeholder.com/1600x800/0d1117/00ff88?text=MATH+SWORDS+VR+%E2%9A%94%EF%B8%8F%E2%9A%94%EF%B8%8F)  
*Fast-Paced • Educational • Addictively Fun*

Two cubes. One correct answer. Two glowing swords.  
Only the math warriors survive.

---

## Key Features
- Dual-wield sword gameplay (Left & Right controllers)  
- Real-time procedural math: +, −, ×, ÷, exponents (2⁵, 3⁴ …)  
- 4 progressive difficulty levels → higher risk = higher score  
- Satisfying VFX explosions (green/orange for correct, fiery red for wrong)  
- Animated target number with pulse & color highlight  
- 3 lives system with heart UI  
- Clean, modular & commented C# code  
- Built for Quest 1/2/3/Pro & Pico – standalone Android XR  
- Zero external packages – pure Unity

## How to Play
1. Look at the big glowing **Target Number**  
2. Two math cubes rush toward you  
3. Slash ONLY the cube that equals the target  
4. Correct → points + epic explosion  
5. Wrong → lose a heart (3 total)  
6. Survive → become legend
